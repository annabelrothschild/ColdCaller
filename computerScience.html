SCIMAPS

Created with two other team mates, SciMaps is a Google Maps-like application to help users navigate Wellesley College's erratically numbered Science Center. Written in Java, this application utilizes Dijkstra's algorithm to determine the shortest path between rooms and a variety of data structures. My role was implementing many of those data structures to create a system for rooms to be read in from a plain text file and connected to one another in a massive map.
